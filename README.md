## 💀 GHOST-SHELL V2.0 💀
**The Ultimate Cross-Platform Penetration Testing Tool Installer.**

Ghost-Shell adalah script otomasi berbasis Python yang dirancang untuk mempercepat setup environment cybersecurity. Dibuat dengan antarmuka terminal modern (Rich UI) untuk kenyamanan maksimal para researcher.

---

## 🔥 Fitur Utama
* **Multi-Platform:** Support Termux (Android), Linux (Ubuntu/Kali), MacOS, dan Windows.
* **Modern UI:** Menggunakan tabel dan panel berwarna untuk pembacaan data yang lebih baik.
* **Smart Detection:** Otomatis mendeteksi sistem operasi dan mengarahkan path instalasi.
* **One-Click Install:** Tidak perlu menghafal URL repository GitHub satu per satu.

## 🛠️ Daftar Tools Tersedia
1. **Nmap** - Network Mapping & Scanning.
2. **SQLMap** - Automated SQL Injection.
3. **Metasploit** - Powerful Exploitation Framework.
4. **Sherlock** - Social Media Account Hunter (OSINT).
5. **Zphisher** - All-in-one Phishing Tool.
6. **SocialPhish** - Modern Phishing Pages.
7. **Ngrok** - Secure Introspection Tunnels.
8. **SocialBox** - Brute Force Attack Framework.
